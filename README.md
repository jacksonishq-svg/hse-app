# HSE App (Health, Safety & Environment)

Aplikasi manajemen observasi HSE berbasis web dengan React.js dan Flask.

## 📋 Fitur

- 🔐 **Autentikasi**: Register, Login, JWT Token
- 👥 **Multi-role**: Admin dan User
- 📊 **Dashboard**: Statistik observasi real-time
- 📝 **Observasi**: Create, Read, Update, Delete
- 🔍 **Filter**: Berdasarkan status, kategori, severity
- 📱 **Responsive**: Mobile-friendly dengan Tailwind CSS

## 🛠️ Teknologi

**Backend:**
- Python Flask
- PostgreSQL
- JWT Authentication
- SQLAlchemy ORM

**Frontend:**
- React.js
- Vite
- Tailwind CSS
- Axios
- React Router DOM
